# Meeting Coach Frontend

React Native frontend for the Teams Meeting Coach application.

## Setup

```bash
npm install
```

## Development

```bash
# Start Metro bundler
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android
```

## Testing

```bash
npm test
```

## Structure

```
src/
├── components/
│   └── MeetingCoachUI.jsx    # Main UI mockup component
└── ...
```
